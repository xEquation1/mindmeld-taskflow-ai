Challenges and SolutionsChallengeSolutionInconsistent AI FormattingInitially, the AI returned meeting summaries as large walls of text. Implemented asynchronous loading states (spinners) to improve UX and provide feedback while the AI processed the request.  Addressing Incorrect SuggestionsThe AI would sometimes hallucinate deadlines. Solved by updating the system prompt to explicitly state: "Only include deadlines that are explicitly mentioned in the text".
